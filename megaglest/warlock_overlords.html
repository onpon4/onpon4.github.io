Warlock Overlords

This is a tactical scenario for MegaGlest. You can install it by extracting the Zip file below to ~/.megaglest/scenarios (or wherever scenarios for your MegaGlest installation are located).

  • Download: warlock_overlords.zip
  • Forum Thread: https://forum.megaglest.org/index.php?topic=9920.0

This is a simple one. You take control of four archmages and have to use them to destroy four settlements.

  • Player faction: Magic
  • Enemy faction: Tech
  • Tileset: Dark Forest
  • Difficulty: Medium
  • Engine: MegaGlest

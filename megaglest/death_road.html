Death Road

This is a tactical scenario for MegaGlest. You can install it by extracting the Zip file below to ~/.megaglest/scenarios (or wherever scenarios for your MegaGlest installation are located).

  • Download: death_road.zip
  • Forum Thread: https://forum.megaglest.org/index.php?topic=9951.0

You must guide a man along a deadly path so that he can reach the castle safely. Unfortunately, his guards are severely outnumbered and outgunned.

  • Player faction: Tech
  • Enemy faction: Magic, Indian
  • Tileset: Dark Forest
  • Difficulty: Hard
  • Engine: MegaGlest

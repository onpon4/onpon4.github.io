Native Unrest

This is a tactical scenario for MegaGlest. You can install it by extracting the Zip file below to ~/.megaglest/scenarios (or wherever scenarios for your MegaGlest installation are located).

  • Download: native_unrest.zip
  • Forum Thread: https://forum.megaglest.org/index.php?topic=9921.0

In this scenario, you take command of a disgruntled Indian tribe. Your mission is to destroy the castle to the north-East, putting an end to the king's reign. But the only path to the king's castle is heavily guarded, and you are outnumbered.

  • Player faction: Indian
  • Enemy faction: Tech
  • Tileset: Winter Forest
  • Difficulty: Hard
  • Engine: MegaGlest
